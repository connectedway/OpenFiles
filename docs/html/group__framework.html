Detailed Description

These APIs are for advanced use cases. Assuming the build has been configured with INIT_ON_LOAD, and OFC_PERSIST has also been configured so that runtime configuration of the stack is restored from the runtime configuration file, there should be no need for an application developer to call any of these routines.
