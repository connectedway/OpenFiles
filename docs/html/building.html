Building the Open Files Stack

To build using CLion, simply run the CLion IDE and use the File Open menu item to open the directory of the openfiles repository. Once opened, it will automatically generate the necessary makefiles for your platform. Then using the Build Project menu item will generate all targets.

To build using cmake directly, first you will need to generate the necessary makefiles. This can be done by running a command shell and issuing the following command:

cmake -DOPENFILE_CONFIG=./configs/default .

To build, simply issue the cmake command with the build argument.

cmake --build .

Output will be written to the source directories. Expected output includes:

  • of_core/libof_core_static.a
  • of_core/libof_core_shared.dylib
  • of_core/libof_core_shared.5.dylib
  • of_core/libof_core_shared.5.0.1.dylib

as well as various test applications in of_core/test

Using other cmake options can place the output in other locations
